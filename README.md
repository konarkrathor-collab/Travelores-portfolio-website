# Travelores - Travel Itinerary Platform 

- Overview
Travelores is a full-stack web application designed to provide users with carefully curated travel itineraries. It features a custom-built front-end for browsing travel plans and a robust Node.js backend connected to a MySQL database to handle data management and user interactions seamlessly. 

- Project Screenshots




---- Tech Stack
* Frontend:HTML5, CSS3, Vanilla JavaScript
* Backend: Node.js
* Database: MySQL
* Key Implementations: Secure database connectivity, client-side/server-side form validation

---- Key Features
* Curated Itineraries: Users can browse through hand-picked, detailed travel plans and destination guides.
* Robust Form Validation: Ensures that all user inputs (such as contact forms or booking inquiries) are accurate and secure before being processed.
* Relational Database Management: Utilizes MySQL to efficiently store, manage, and retrieve itinerary details and user data.
* Full-Stack Architecture: Smooth integration between the interactive UI and the Node.js server.

-> How to Run Locally
1. Clone the repository: `git clone https://github.com/konarkrathor-collab/Travelores-portfolio-website.git`
2. Navigate to the backend directory: `cd travelores-backend`
3. Install dependencies: `npm install`
4. Set up your MySQL database using the provided schema.
5. Start the server: `node index.js` *(or `npm start`)*
